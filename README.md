This repository provides dynamic DNN configurations for branchy model with early-exits and an energy efficient fraemwork to orchestrate such a model in a distributed network system.

Please cite the following papers, for this codebase:

1. C. Singhal, Y. Wu, F. Malandrino, M. Levorato and C. F. 2. C. Singhal, Y. Wu, F. Malandrino, M. Levorato and C. F. Chiasserini, "Distributing Inference Tasks Over Interconnected Systems Through Dynamic DNNs," in IEEE Transactions on Networking, vol. 33, no. 4, pp. 1717-1730, Aug. 2025, doi: 10.1109/TON.2025.3543848.
